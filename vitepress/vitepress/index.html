电脑安装 VitePress 详细教程

📅 2025-12-10 📁 vitepress 🏷️ vitepress

VitePress 是基于 Vite 构建的静态网站生成器，专为技术文档设计，具有极速启动、热更新、优化的构建性能等优势。本文将详细介绍在 Windows、macOS、Linux 系统上安装 VitePress 的完整步骤，适用于新手入门。

一、前置条件

VitePress 依赖 Node.js 环境，因此在安装前需确保电脑已安装Node.js 18.0 或更高版本（推荐使用 LTS 长期支持版，稳定性更强）。

1. 检查 Node.js 是否安装

打开电脑的终端（Windows 用 CMD 或 PowerShell，macOS/Linux 用 Terminal），输入以下命令检查 Node.js 和 npm（Node.js 自带的包管理器）版本：

若输出类似 v20.10.0（Node.js 版本）和 10.2.3（npm 版本）的信息，说明已安装；若提示“不是内部或外部命令”，则需要先安装 Node.js。

2. 安装/更新 Node.js

前往 Node.js 官方网站https://nodejs.org/，下载对应系统的 LTS 版本安装包：

  • Windows：下载 .msi 安装包，双击运行，勾选“Add to PATH”（自动添加环境变量），后续按默认选项下一步即可。

  • macOS：下载 .pkg 安装包，双击运行按提示安装；或使用 Homebrew 安装（终端输入 brew install node@20）。

  • Linux：可通过包管理器安装，例如 Ubuntu 输入 sudo apt install nodejs npm，若版本过低，可参考官方文档升级。

安装完成后，重新打开终端，再次执行 node -v 和 npm -v 验证安装成功。

五、常见问题解决

1. “node 不是内部或外部命令”

原因：Node.js 安装时未添加环境变量。解决方法：重新运行 Node.js 安装包，勾选“Add to PATH”；或手动配置环境变量（参考 Node.js 官方文档）。

2. 安装依赖时卡住或失败
